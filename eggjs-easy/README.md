# eggjs

# 开发

```bash
$ npm i
$ npm run dev
$ open http://localhost:7001/
```

# Deploy

```bash
$ npm start
$ npm stop
```

3. 执行 f deploy

> 2.0
 
1. npm i @midwayjs/cli -g // @midwayjs/cli 提供了两个入口命令。 midway-bin  和 mw  命令。

> 3.0

1. 不再提供部署，midwayjs只提供框架和本地开发能力。
